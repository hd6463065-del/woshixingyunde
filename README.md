第 1 页：モジュール構成（模块构成）
画面モジュール概要
本画面は、法人 DM システムにおける基礎データの一括補正機能を提供する。ユーザーが Excel ファイルをアップロードし、データチェックを経て、「基礎データ補正管理」テーブルにデータを登録する。モジュールは、ファイルアップロード、データチェック、エラー表示、データ登録の 4 つのサブモジュールから構成される。
画面モジュール一覧
表格
Noプログラムファイル名処理概要
1 main.py 画面全体の制御、初期表示、イベントハンドリングを行う 
2 message_popup.py 共通メッセージ部品：メッセージのキューイングと画面表示を担当 
3 message_data.py メッセージ ID と文言のマッピング定義 
4 file_validator.py アップロードファイルのチェック処理（必須項目、データ型、キー重複等） 
5 db_handler.py 「基礎データ補正管理」テーブルへの登録・更新処理を担当
